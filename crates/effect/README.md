## Functionality

- apply an `Effect` on a `rusvid_core::plane::Plane`
- effect library
  - BoxBlur
  - ColorPaletteEffect
  - GaussianBlur
  - GrayscaleEffect
  - PixelateEffect
  - ScriptingEffect (needs feature `scripting` enabled)

## Dependencies

- rustc 1.63.0-nightly
